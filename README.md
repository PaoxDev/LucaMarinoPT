
# 💼 Luca Marino PT – Landing Page
A professional landing page designed for a personal trainer to showcase services, improve online presence, and attract potential clients.

---

## 📖 What is this project?
This project is a **frontend landing page** created for a personal trainer (Luca Marino).
The goal is to provide a clean, modern, and engaging website that promotes fitness services and improves user interaction.

The page is designed to:
* present the trainer professionally
* highlight services and benefits
* encourage users to take action (contact / sign up)

---

## 🎯 Purpose of the Project
* Build a real-world **business-oriented website**
* Improve **UI/UX design skills**
* Practice **responsive layout design**
* Create a **conversion-focused landing page**

---

## 🚀 Technologies Used

This project was developed using:
* **HTML5** → page structure and semantic layout
* **CSS3** → styling, layout, and responsive design
* **JavaScript** → interactivity and dynamic elements

---

## ⚙️ Features
* 💪 Professional fitness landing page
* 📱 Fully responsive design (mobile-friendly)
* 🎯 Call-to-action sections for user engagement
* ⚡ Smooth user experience with clean UI
* 🎨 Modern layout focused on branding

---

## 💡 How it works
This is a **static website**, meaning:
* No backend or database is required
* Runs entirely in the browser
* Fast loading and lightweight

---

## 📦 Requirements
To run the project locally:
* A modern browser (Chrome, Edge, Firefox)

Optional:
* Local server (e.g. Live Server)
